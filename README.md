 IoT-Based Motion Detector using ESP32-CAM

## Overview
This project is an IoT-based motion detection system using ESP32-CAM that captures images when motion is detected. It is designed for basic surveillance and monitoring purposes.

## Features
- Motion detection using PIR sensor  
- Image capture with ESP32-CAM  
- Simple and low-cost setup  

## Components Used
- ESP32-CAM  
- PIR Motion Sensor  
- Jumper wires and power supply  

## Working
When motion is detected by the PIR sensor, the ESP32-CAM captures an image. The system can be used for basic monitoring and security applications.

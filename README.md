# IoT-Based Motion Detector using ESP32-CAM

## Overview
This project is an IoT-based motion detection system using ESP32-CAM that captures images when motion is detected and sends alerts for basic surveillance.

## Features
- Real-time motion detection
- Image capture using ESP32-CAM
- Alert system for monitoring
- Low-cost and efficient setup

## Components Used
- ESP32-CAM
- PIR Motion Sensor
- Jumper Wires
- Power Supply

## Working
The PIR sensor detects motion and triggers the ESP32-CAM to capture an image. The system can send alerts or store captured images for monitoring purposes.

## Applications
- Home security
- Office monitoring
- Basic surveillance systems
